# 高位、低位运算
## 读写
## 1. fp32->bf16
### 1.1 vfcvt.high.bf16.fp32.v v1,v2
将fp32的v2的高位（后一半）转换成bf16的v1
例：
v2=[... b8c6 4410 16fd 42b2]，所以fp32的v2的第一个数是0x4410b8c6，
- top16 = 0x4410
- low16 = 0xb8c6 （> 0x8000，所以要进位）
- lsb = top16 & 1 = 0
- bf16 = (0x4410b8c6 + 0x7FFF) >> 16 = 0x4411
同理第二个数 0x42b216fd的低位是low16=0x16fd （<0x8000，不进位），所以高位是42b2
所以v1=[4411 42b2]
### 1.2 vfcvt.low.bf16.fp32.v v1,v2
将fp32的v2的高位（前一半）转换成bf16的v1
v2=[b8c6 4410 16fd 42b2]，
v1=[4411 42b2]

## 2. bf16->fp32
### 2.1 vfcvt.high.fp32.bf16.v v1,v2

v1=[0000 440e 0000 c3e9]

将bf16的v2的高位(前一半)转换成fp32的v1
