# 高位、低位运算
## 读写
### 1. vfcvt.high.bf16.fp32.v v1,v2
将fp32的v2的高位转换成bf16的v1

例：
v2=[b8c6 4410 16fd 42b2]，所以fp32的v2的第一个数是0x4410b8c6，
- top16 = 0x4410
- low16 = 0xb8c6 （> 0x8000，所以要进位）
- lsb = top16 & 1 = 0
- bf16 = (0x4410b8c6 + 0x7FFF) >> 16 = 0x4411
同理第二个数 0x42b216fd的
